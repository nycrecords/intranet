My computer won't start.

  1. Check the power cord on the computer
    1. Examine the power cord to make sure that it is plugged in securely at both ends.
    z
  2. Check the monitor
    1. It could be that the computer is on, but the monitor is off.
    2. Check the power cord on both ends and video cable (VGA or Display port) on the monitor.
  3. If both the computer and monitor cannot start, it is most likely an issue from the power strip or surge protector
    1. Ensure that they are plugged in.
    2. Check the on/off switch on the power strip or surge protector.
    3. If there is too much load on the power strip, try to reduce the load.

I can't access the network or Internet.

  1. Check to see if anyone else around you is having a similar problem. If so, there may be a service outage affecting a wider area. Please inform the support staff or help desk.
  2. Check to make sure the network cable is connected to both the computer and the wall.
  3. Check where the network cable connection is made to the back of the computer. You should see a little green or orange light right where the cable connects. If the light isn’t on or flashing, then you are not getting a signal to the computer. If rebooting does not fix it, contact the help desk.

I'm getting an error message.

  1. Take a screenshot and email it to the support staff.
  2. If you experience the BSOD (Blue Screen of Death), use your cell phone to take a picture and send it to the support staff.

My computer is frozen.

If the computer is on but everything on the screen is frozen and the keyboard and mouse are not responding, try tapping on the Num Lock key. It’s located on the right-hand side of the keyboard above the number 7. While tapping the Num Lock key, notice whether or not the Num Lock light goes on and off. If it doesn't, the computer is completely locked up. You’ll have to REBOOT the computer. If the light does go off and on, then wait a minute before giving up hope. It might come back to life on its own.

Pressing Alt + F4 keys - If a software program stops working or freezes up, try pressing the ALT+F4 keys to close the window that you’re currently working in. This can shut down a frozen window and bring the computer back to life.

If Alt + F4 keys do not work, press Ctrl + Shift + Esc keys to open Task Manager, and try to end the software that is not responding.

If the keyboard is frozen as well, you will have to reboot the computer.

I can only see a blue screen.

If the dreaded “Blue Screen of death” (BSOD) has appeared, there is not a lot to be done except to reboot the machine. If it keeps occurring, this is an indication of a major problem with the computer. Use your cell phone to take a picture or use a pen and paper to mark down the error code.

The keyboard won't work.

Swap the keyboard or try plugging the keyboard into another computer. If it doesn't work on the other computer, there is likely something wrong with the keyboard. If it does work on the other computer, there is likely something wrong with the computer or the connection to the computer. Please contact the support staff.

What is CTRL-ALT-DEL?

When you press the CTRL-ALT-DEL buttons together, it will bring up a control panel where you can select “Task Manager” and see if any of the programs are shown as “not responding”. You may be able to shut down the offending program from there.

If you have to change your password, you can use CTRL-ALT-DEL, then select “Change a password”.

If you can't get to this screen, you'll have to reboot the computer.

Should I reboot my computer?

Rebooting can do wonders. Don’t just pull the plug!

A simple reboot may clear up the problem. Go to the Start Menu and select Shutdown. Didn't work? It's time to try a forced reboot — with, unfortunately, no way to save your work — by pressing the keys CTRL-ALT-DEL simultaneously twice in a row.

If that didn't work, you have no choice: you have to turn the computer off manually with the power switch. Hold the power button down (for about 5 -10 seconds) and the computer will shut down. Wait at least 10 seconds for the hard drive to stop spinning and then restart the machine.

If the computer starts up, start the applications you were using when the crash occurred. Some programs, like Microsoft Word, make timed backups of your work and may bring up recovered files. If the program notifies you that there is a “recovered document” save the file under a different name and compare them to your previously saved files.

How do I access archived emails in Enterprise Vault?

DoITT has provided the following information regarding accessing Enterprise Vault and archiving in Office 365:

Background: Before the migration to Office 365, the archival platform for emails has been the Enterprise Vault (EV), an on-premises solution hosted at DoITT. After DoITT migrated all Mailboxes to the cloud (Office 365) then the archiving platform also moved to the cloud. DoITT is working on a project to migrate the legacy data from on-premises Enterprise Vault to Office 365. We do not have a date as to when that project will be completed. However, the following information will help you regarding the archival process.

What is the archiving process in the cloud (Office 365)?

The archiving is done after 1 year in the cloud (Office 365) since the primary mailbox limit is 100 GB per mailbox and the archiving is unlimited. This eliminates the on-premises archiving policy as the storage is not an issue anymore. DORIS migrated to Office 365 as of February 26th, 2020. So, the archiving began after that date in the year 2021 for us.

How do I forward Phishing emails to NYC3 (phish@cyber.nyc.gov)?

  1. With the suspected phishing email highlighted or opened, hit CTRL-ALT-F. (this will forward the email, with required tracking information (headers), as an attachment, to the NYC Cyber Command Security Operations Center for analysis.)
  2. Enter phish@cyber.nyc.gov as the recipient and click send.
  3. Delete the suspicious email from your mailbox without clicking on any hyperlinks or attachments and delete it from the Deleted Items folder.
